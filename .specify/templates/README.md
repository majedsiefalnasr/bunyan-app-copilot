# .specify/templates

> **Legacy compatibility surface.**

The canonical templates are located in `specs/templates/`.

This directory exists for backward compatibility with tools and scripts that reference `.specify/templates/`.

See:

- `specs/templates/` — All spec, plan, task, commit, report, audit, and guide templates
- `specs/templates/commits/` — Commit message templates per step
- `specs/templates/reports/` — Report templates per step
- `specs/templates/audits/` — Audit and validation report templates
- `specs/templates/guides/` — Testing and implementation guides
